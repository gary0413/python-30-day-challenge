# 30天Python學習計畫

這是一個為期 30 天的 Python 學習計畫，專為初學者設計。計畫從最基本的概念開始，逐步深入，並強調每日實作，確保您能打下穩固的基礎。

### 成功的關鍵心法
1.  **每日實踐 (Practice Daily):** 每天至少花 1-2 小時。寫程式是技能，需要持續練習。
2.  **動手做，不只是看 (Code, Don't Just Read):** 親手把每個範例打一次，然後試著修改它、玩玩看。
3.  **遇到錯誤是正常的 (Embrace Errors):** 錯誤訊息是學習的一部分。試著讀懂它，或利用搜尋引擎尋找答案。
4.  **建立小專案 (Build Things):** 將所學應用到小專案中，是鞏固知識最好的方法。

---

### **第一週：Python 基礎與核心邏輯**

*   **第 1 天：環境設定與第一個程式**
    *   **學習:** 安裝 Python 和 VS Code (一個流行的程式編輯器)。
    *   **實作:** 寫出您的第一個程式：`print("Hello, World!")`，並成功執行它。理解什麼是**變數 (Variable)**。

*   **第 2 天：基本資料型態與運算**
    *   **學習:** 字串 (String), 整數 (Integer), 浮點數 (Float)。
    *   **實作:** 練習變數賦值、數學運算 (`+`, `-`, `*`, `/`) 和字串拼接。

*   **第 3 天：使用者輸入與型態轉換**
    *   **學習:** 使用 `input()` 函數從使用者那裡獲取資料。
    *   **實作:** 寫一個簡單的互動程式，例如詢問使用者的名字並向他問好。練習使用 `int()`、`str()` 進行型態轉換。

*   **第 4 天：串列 (List) 的基本操作**
    *   **學習:** 什麼是 List？如何建立、存取、新增和刪除 List 中的元素。
    *   **實作:** 建立一個購物清單，並對其進行新增、刪除、檢視等操作。

*   **第 5 天：條件判斷 (Conditional Logic)**
    *   **學習:** `if`, `elif`, `else`。布林值 (`True`, `False`) 和比較運算子 (`==`, `!=`, `>`, `<`)。
    *   **實作:** 寫一個程式，判斷使用者輸入的數字是正數、負數還是零。

*   **第 6 天：For 迴圈 (For Loops)**
    *   **學習:** 如何使用 `for` 迴圈遍歷 (iterate) 一個 List。
    *   **實作:** 將購物清單中的每個項目逐一印出。

*   **第 7 天：While 迴圈與本週複習**
    *   **學習:** `while` 迴圈的用法。
    *   **實作:** 寫一個簡單的猜數字遊戲。複習本週所有概念。

---

### **第二週：資料結構與進階迴圈**

*   **第 8 天：串列的進階操作**
    *   **學習:** List 切片 (Slicing)、排序 (`.sort()`)、反轉 (`.reverse()`)。
    *   **實作:** 練習對數字串列進行排序和切片。

*   **第 9 天：字典 (Dictionary)**
    *   **學習:** 什麼是字典 (key-value pairs)。如何建立、存取、新增和修改字典。
    *   **實作:** 建立一個儲存個人資訊（姓名、年齡、城市）的字典。

*   **第 10 天：遍歷字典**
    *   **學習:** 如何使用 `for` 迴圈遍歷字典的 keys, values, 和 items。
    *   **實作:** 將個人資訊字典中的每個鍵值對都印出來。

*   **第 11 天：元組 (Tuple) 與集合 (Set)**
    *   **學習:** Tuple (不可變的 List) 和 Set (無序且元素唯一的集合) 的特性與使用時機。
    *   **實作:** 練習建立 Tuple 和 Set，並嘗試對 List 使用 `set()` 來移除重複項。

*   **第 12 天：巢狀資料結構 (Nested Data Structures)**
    *   **學習:** 如何在 List 中包含 Dictionaries，或在 Dictionary 中包含 Lists。
    *   **實作:** 建立一個班級名單 (List)，其中每個學生都是一個 Dictionary。

*   **第 13-14 天：專案練習與複習**
    *   **專案:** 寫一個簡易的聯絡簿程式。使用者可以新增、查詢、刪除聯絡人資訊。
    *   **複習:** 複習本週所有資料結構。

---

### **第三週：函式、模組與檔案處理**

*   **第 15 天：函式 (Functions)**
    *   **學習:** 如何使用 `def` 定義自己的函式，以及為什麼要使用函式。
    *   **實作:** 將之前寫的程式碼（如猜數字遊戲）改寫成使用函式的版本。

*   **第 16 天：函式的參數與回傳值**
    *   **學習:** 函式的參數 (Parameters) 和回傳值 (`return`)。
    *   **實作:** 寫一個接收兩個數字作為參數並回傳它們總和的函式。

*   **第 17 天：變數作用域 (Scope)**
    *   **學習:** 區域變數 (Local) vs. 全域變數 (Global)。
    *   **實作:** 透過範例理解兩者的差異。

*   **第 18 天：模組 (Modules)**
    *   **學習:** 如何使用 `import` 匯入 Python 內建模組，例如 `math` 和 `random`。
    *   **實作:** 使用 `random` 模組來改進猜數字遊戲。

*   **第 19 天：錯誤處理 (Error Handling)**
    *   **學習:** 使用 `try...except` 來捕捉和處理程式執行時的錯誤。
    *   **實作:** 在要求使用者輸入數字的程式中，加入錯誤處理以防止使用者輸入非數字字元。

*   **第 20 天：讀取檔案**
    *   **學習:** 如何使用 `with open(...)` 來讀取 `.txt` 檔案的內容。
    *   **實作:** 建立一個文字檔，並寫程式讀取其內容並印出。

*   **第 21 天：寫入檔案與本週複習**
    *   **學習:** 如何寫入 (`'w'`) 和附加 (`'a'`) 內容到檔案中。
    *   **實作:** 將聯絡簿的資料儲存到檔案中，並能從檔案讀取回來。

---

### **第四週：物件導向、外部套件與綜合應用**

*   **第 22-23 天：物件導向程式設計 (OOP) - 類別與物件**
    *   **學習:** 什麼是類別 (Class) 和物件 (Object)。如何定義一個類別，包含 `__init__` 建構函式和屬性 (Attributes)。
    *   **實作:** 建立一個 `Dog` 類別，可以實例化 (instantiate) 出不同的狗狗物件。

*   **第 24 天：類別的方法 (Methods)**
    *   **學習:** 如何在類別中定義方法 (即屬於該類別的函式)。
    *   **實作:** 為 `Dog` 類別新增 `bark()` (吠叫) 和 `sit()` (坐下) 等方法。

*   **第 25 天：使用 PIP 安裝外部套件**
    *   **學習:** 什麼是 PIP (Python 的套件管理器)。
    *   **實作:** 學習 `pip install` 指令，並安裝一個有趣的套件，例如 `requests` (用來發送網路請求)。

*   **第 26 天：與 API 互動**
    *   **學習:** 什麼是 API。使用 `requests` 套件來獲取一個公開 API 的資料 (例如天氣、笑話等)。
    *   **實作:** 寫一個程式，從 API 獲取資料並將其印出。

*   **第 27-28 天：處理 JSON 資料與複習**
    *   **學習:** 什麼是 JSON 格式。如何使用 `json` 模組來解析 API 回傳的 JSON 資料。
    *   **實作:** 解析從 API 獲取的 JSON 資料，並只取出您需要的部分。

*   **第 29-30 天：總結專案 (Capstone Project)**
    *   **目標:** 綜合運用這 30 天所學，獨立完成一個小專案。
    *   **專案點子:**
        1.  **命令列待辦事項清單 (To-Do List App):** 讓使用者可以新增、檢視、完成、刪除待辦事項，並將清單存檔。
        2.  **簡易天氣查詢工具:** 讓使用者輸入城市名稱，程式透過 API 查詢並顯示該城市的即時天氣。
        3.  **個人密碼管理器:** (進階) 讓使用者可以儲存不同網站的帳號密碼 (注意：僅為學習目的，請勿儲存真實密碼)。
